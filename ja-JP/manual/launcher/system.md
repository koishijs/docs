# システム要件

このセクションでは、Koishi ランチャーがサポートする OS のバージョンと機能を説明します。

:::tip
以下のリストは Koishi ランチャーにのみ有効です。Koishi 自体はほとんどの主流の環境で実行できます。
:::

:::tip
あなたのオペレーティングシステムが Koishi 公式にサポートされなくても、[フォーラムで](https://k.ilharp.cc) コミュニティが提供する互換バージョンを選び、または [テンプレートプロジェクトを作成する](../starter/boilerplate.md) ことにしてもいいです。ただ、Koishi 公式はコミュニティ ランチャーの可用性と安全性について責任を負いません。
:::
