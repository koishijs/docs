# 認識控制檯

Koishi 的一個核心特性是強大的控制檯。控制檯是一個對使用者非常友好的圖形介面，封裝了 Koishi 的絕大多數功能：

- 狀態監控、資料統計
- 配置外掛
- 安裝、解除安裝和更新外掛
- 管理指令、資料庫、輸出文字
- 模擬聊天
- 檢視日誌

## 控制檯介面

在您成功安裝了模板專案或啟動器後，控制檯將自動開啟。

控制檯的介面主要分為兩部分，左側是活動欄，用來切換右邊的介面，右邊預設開啟的是儀表盤頁面。在寬屏裝置的下方還有一個狀態列，主要用來顯示機器人的執行狀態。
